A movie ticket booking system that allows users to browse and book show tickets across cities.
Core Requirements:
List all cities where theaters are present.
On city selection, display movies running in that city.
After movie selection, show available shows (screens, date, time, status).
Allow users to view seating layout and select seats.
Enforce a maximum seat limit per transaction.
Tickets are booked in FIFO (First-In-First-Out) order.
Confirm bookings via SMS, Email, or WhatsApp notifications.
